# College Football Team Prediction App

This project is a machine learning web app that predicts team performance and postseason outcomes.

## Features
- Forward prediction (availability + win % → outcomes)
- Reverse prediction (outcomes → estimated inputs)
- Interactive web interface

## Notes
- Predictions are based on historical data
- Reverse predictions are approximate
- Results should be interpreted as estimates, not guarantees

## Tech Stack
- Python
- Streamlit
- Pandas
- Scikit-learn
